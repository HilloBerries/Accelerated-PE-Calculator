Accelerated PE Calculator is a TI-84 calculator program with useful tools for weight lifting for a school that bans phones.

[Install Instructions]:
1. Add to RAM through the TI Connect software or an alternative
2. Make sure ACCELV1.8xi is Pic0, or the splash image will fail to draw
3. Run through the PRGM menu

[Included Tools]:
1. Warm Up Calculator
2. Plate Calculator
